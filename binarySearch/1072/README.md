#문제 풀이
링크 
백준 1072문제 게임 (링크 : https://www.acmicpc.net/problem/1072)

## 제한조건
시간제한 2초
메모리제한 128MB

## 시간 복잡도 분석 및 전략
단순 선형 탐색시 기본 입력값이 10억이기 때문에 선형 으로 케이스 계산시 시간 초과 가능성존재 O(N)
이분탐색의 매개변수 서치를 통해 시간복잡도 줄이는 방법 고려  O(logN)

이분 탐색으로 구현시
이분 탐색 사용 시: 한 배열을 정렬 O(logR), 정렬된 데이터 안에서 특정 값이 어디 있는지 찾는 것
10억인 경우에도 30번의 연산으로 해결가능

### 해결방안
시간 복잡도 전략: 매개변수 서치를 통해 정답이 있다라는걸 가정하에 해당 값을 계속 최소값 이 나오게 연산 

### 예외케이스
기존 확률 이 변하지 않는경우 승률이 100% 또는 99%인 경우에는 연산 처리 불가 이부분에 대한 예외처리 필요
확률 계산시 int 범위를 벗어날수 있으니 자료형을 long으로 받아야함
