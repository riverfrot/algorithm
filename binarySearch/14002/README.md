#문제 풀이
링크 
백준 14002문제 (링크 : https://www.acmicpc.net/problem/14002)

## 제한조건
시간제한 1초
메모리제한 256MB

## 시간 복잡도 분석 및 전략
DP로도 O(N^2)으로 풀이가 가능하나 이분탐색으로 O(NlogN) 구현(LIS)
100만 -> 1만 정도로 시간복잡도 효율화
추가로 기존 문제(11053)과 달리 백트랙킹 로직도 추가가 필요
O(NlogN)+O(N) => O(NlogN)으로 최종 시간복잡도를 확인가능

### 해결방안
LIS를 통해 기존값보다 같거나 작으면 왼쪽에 있는항목을교체 크다면 오른쪽 arrayList를 키우는 방식으로 구현 
문제에서 가장 긴 증가하는 부분 수열을 구하라 하였으므로 위에 대해서 백트랙킹으로 해당 수열을 찾아가는 로직이 필요

### 예외케이스
기존 문제 11053과 달리 수열(이어진숫자)에 대한 정답을 구하는 연산이추가적으로 필요

### 고민했던 흔적들
기존 문제 11053과 달리 수열(이어진숫자)에 대한 정답을 구하는 연산이추가적으로 필요
이에따라 백트랙킹을 통해 연산을 추가하였으나 혹시 다르게 접근할수있는방법이 있을까 고민
