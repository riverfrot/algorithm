#문제 풀이
링크 
프로그래머스 최소직사각형 (링크 : https://school.programmers.co.kr/learn/courses/30/lessons/86491)

## 제한조건
sizes의 길이는 1 이상 10,000 이하입니다.
sizes의 원소는 [w, h] 형식입니다.
w는 명함의 가로 길이를 나타냅니다.
h는 명함의 세로 길이를 나타냅니다.
w와 h는 1 이상 1,000 이하인 자연수입니다.

## 시간 복잡도 분석 및 전략
시간복잡도 : O(N)

### 해결방안
선형 탐색

### 예외케이스

### 고민했던 흔적들
문제의 조건이 어느한쪽을 크게 만들고 나머지 배열에 대해서 
정렬 후 가장 큰 요소를 배치 하는것이 가능하므로 일단 가장 큰 값을 찾아서 해당 영역으로 다 위치를 바꿨을때
해당 값이 남은 수 중에 가장 큰수를 곱하면됨.

일단 입력시에 가장 큰값을 찾고 그게 0요소인지 1번요소인지 확인
가장 큰값과 가장 작은값을 기준으로 (가장작은값은-> 추후 swap 할것이므로) 별도로 정렬하지않아도 
바로 확인가능
