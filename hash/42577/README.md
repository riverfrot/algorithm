## 제한조건

## 시간 복잡도 분석 및 전략
입력값이 10만인 배열 두개가 들어오고 이 두개의 배열중하나의 배열중에 하나의 배열에서 남는 데이터를 찾아야함
이경우 단순 선형 방식으로 이중 반복문을 시도하게 되면 O(N^2)의 시간 복잡도를 갖게되고 100억 케이스를 테스트해야 문제를 해결가능
이떄 해쉬라는 구조를 사용해 같은 키를 탐색할때 O(1)의 시간을 갖는 풀이 방법을 사용
총 3개의 반복문을 순회 하게되어 최종 시간 복잡도는 O(N)이 됨
