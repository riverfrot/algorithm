#문제 풀이
링크 
프로그래머스 가장큰수 (링크 : https://school.programmers.co.kr/learn/courses/30/lessons/42746)

## 제한조건
제한 사항
numbers의 길이는 1 이상 100,000 이하입니다.
numbers의 원소는 0 이상 1,000 이하입니다.
정답이 너무 클 수 있으니 문자열로 바꾸어 return 합니다.

## 시간 복잡도 분석 및 전략
시간 복잡도 (정렬 이용) : O(NlogN)
공간 복잡도 : O(N)

### 해결방안
정렬과 복사를 사용
